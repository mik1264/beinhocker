# Kauffman's Adjacent Possible

## Definition
The adjacent possible is the set of all configurations that are exactly one step away from the current state of a system. It represents what is reachable through a single modification.

## Key Properties
- **Bounded**: Not everything is possible, only things one step away
- **Expanding**: Each exploration of the adjacent possible reveals new adjacent possibles
- **Path-dependent**: The current adjacent possible depends on history
- **Recursive**: Opening one door reveals new doors

## In Boolean Networks
- The adjacent possible of a network state = all states at Hamming distance 1
- Each state has exactly N neighbors in the state space
- Adaptation explores the adjacent possible by flipping one bit at a time
- The fitness landscape determines which adjacent states are improvements

## Organizational Application
- Innovation as exploration of the adjacent possible
- Organizations can only reach configurations near their current state
- Radical innovation = reaching states far from current position (requires multiple steps)
- Strategy = choosing which parts of the adjacent possible to explore

## Design Space Implications
- The adjacent possible constrains what innovations are feasible
- Technologies build on each other (adjacent possible expands over time)
- First-mover advantage = exploring adjacent possible before competitors
- Platform strategies = expanding the adjacent possible for others
