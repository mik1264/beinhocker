# Autocatalytic Sets and Kauffman's Origin of Order

## Kauffman's Theory (1971, 1993)
- Autocatalytic sets: collections of molecules that mutually catalyze each other's formation
- Self-sustaining given a basic "food" source
- Emergence is essentially inevitable in sufficiently complex chemical systems
- Self-organization plays a role alongside Darwinian selection

## Reflexively Autocatalytic Food-Generated Networks (RAFs)
- Self-sustaining networks that collectively catalyze all their reactions
- Can be identified embedded within real metabolic networks
- Phase transition: autocatalytic set emergence is a threshold phenomenon

## Connection to Jain-Krishna Model
- Jain-Krishna builds on Kauffman's autocatalytic set concept
- Network evolves through species replacement, creating and destroying ACS
- ACS emergence triggers exponential connectivity growth
- ACS destruction triggers mass extinction events
